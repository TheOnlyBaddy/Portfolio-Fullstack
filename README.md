## âš™ï¸ Installation & Setup

Clone the repository:
```
git clone https://github.com/TheOnlyBaddy/Portfolio.git
cd Portfolio
```
Install dependencies:
```
npm install
```
Create a .env file:
```
PORT=5000
MONGO_URI=mongodb://127.0.0.1:27017/contactdb
EMAIL_USER=youremail@gmail.com
EMAIL_PASS=your_app_password
OWNER_EMAIL=youremail@gmail.com
```
Run the server:
```
npm start
```
Now open the site in your browser:
```
http://localhost:5000
```
